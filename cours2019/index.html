Data types

A data type defines the operations that can be done on the data, the meaning of the data, and the way values of that type can be stored.

Primitive data types

  • Boolean type
  • Numeric type (integer, floating point, biginteger)
  • Character (String) type

Enumerated data types

Set of values (e.g. CLUB, DIAMOND, HEART, SPADE)


		enum Suit {CLUB, DIAMOND, HEART, SPADE}; 
	    

		data Suit = Club | Diamond | Heart | Spade

Parametrized data types

  • array of <type>, pointer on <type> 
    int tab[5]; int* ptr;
  • list of <type> 
    ArrayList<Integer> lst;
  • function with a given type signature
    func :: Integer -> Integerint (*funcPtr)(int);

implies (static) polymorphism

Composite data types

  • Arrays, lists, ... 
    int tab[5];
  • Records (struct, tuple)
    struct pairIntChar { int first; char second };
  • Classes/objects (contains fields and methods)

Hierarchy of classes and dynamic linking


		class ShopKeeper(Human):
		  def askPaiement(self, x):
		    print("give me " + str(x) + " euros, please")
	    

		fred = ShopKeeper()
		fred.sayHello()
		fred.askPaiement(10)
	    

	    >>> me.askPaiement(10)
	    Traceback (most recent call last):
	    File "", line 1, in 
	    AttributeError: 'Human' object has no attribute 'askPaiement'
